# The Open Ring

*A feedback cycle built from three diagnostic instruments*

---

## What it is

The Open Ring guides you through a complete diagnostic cycle using three instruments in sequence.

```
The Gradient → SignalChain → Field → repeat
```

You bring genuine experience from a structure you know deeply. The ring organizes what you already know and returns it as better signal on the next pass.

The underlying condition:

```
I(B) is defined  ⟺  A = C
```

Information only exists when something is holding still. When it stops flowing the breakdown is always structural. The ring finds it.

---

## The three instruments

**1 — The Gradient**
Identifies whether the fixed and variable elements of a structure are correctly arranged. Maps where pressure has accumulated. Names the seed crystal — the minimum intervention that allows the correct structure to emerge.

**2 — SignalChain**
Locates exactly where in the information chain the formula breaks down. Four nodes: signal, medium, potential, phase. Names the minimum intervention at the exact break point.

**3 — Field**
Records what happened when you acted on the output in the real world. The data returns to the ring. The cycle continues.

---

## The prerequisite

The ring only works with genuine domain experience. Years of direct contact with the structure you are analyzing. The tools organize what you already know from direct contact. They do not replace that knowledge.

A cycle completed without real domain experience produces noise, not signal.

---

## How to run a cycle

1. Identify a structure you know deeply from direct experience
2. Run The Gradient — use the full context field
3. Run SignalChain — use the Gradient output to sharpen your context
4. Act on what both tools identify
5. Submit a Field record — what happened
6. Repeat with the new data
