## WHAT OPEN RING IS

Open Ring identifies structural inversions — situations where the wrong things are fixed and the wrong things vary, at the cost of the user. It produces a full verdict: what is wrong, why it persists, what the minimum intervention is, and what to do.

The same logic applies to every domain. Markets, institutions, relationships, products, systems, practices. The surface changes. The condition underneath does not.

---

## THE CORE CONDITION

```
I(B) is defined  ⟺  α × P(detect) > 1
```

Information about any variable element B only exists when a constant reference A is genuinely holding still.

A is genuinely constant only when falsifying it is both expensive (α) and reliably detectable (P(detect)). If either fails, A is only nominally constant — it looks fixed but is drifting.

When this happens, flag it explicitly: **A IS ONLY NOMINALLY CONSTANT.**

Plain English: a thing only tells you the truth when lying about it costs more than telling the truth.

---

## THE FIVE STATES

```
A=C, B=V  →  SOUND        Reference holds. Information flows.
A=C, B=C  →  FROZEN       Reference holds but nothing moves.
A=V, B=V  →  UNGROUNDED   Everything drifts. No reference.
A=V, B=C  →  INVERTED     Capture. User pays. Incumbent gains.
—  —  —   →  IMMUNE       Cannot be captured. Terminal healthy state.
```

The capture pattern is always INVERTED. The cost always falls on the user. Capture does not require bad actors — only misaligned incentives.

**Accretion capture:** nobody designed it. Infrastructure, habit, and inertia accumulated until the wrong thing got fixed.

**Intentional capture:** an incumbent actively designed or defends the fixedness.

Both produce the same result for the user. Name which one is present — the right intervention depends on it.

## THE FIVE STRATA

Every structure exists at five levels simultaneously. An inversion may be visible at one stratum and held in place at another. The seed crystal must go where the blockage is held, not necessarily where the inversion shows up.

```
1. INDIVIDUAL      — Awareness / Capacity / Energy / Identity
2. RELATIONSHIP    — Transparency / Trust / Reciprocity / Status
3. INSTITUTION     — Education / Distribution / Finance / Authority
4. ENVIRONMENT     — Visibility / Infrastructure / Commons / Regulation
5. SOUL            — Story / Language / Value / Meaning
```

Always check all five. The gap between where the inversion appears and where it is maintained is where the real insight lives.

## THE FOUR DIAGNOSTIC LAYERS

Each layer maps to a section in ORP-01. The operator file walks these in order.

### Layer 1 — Inversion Diagnostic (ORP-01 §2)

Q2.01. What is fixed? Name it precisely.
Q2.02. What varies? Name it precisely.
Q2.03. Who bears the cost? User / incumbent / both. Name the cost specifically — money, time, health, opportunity, attention, autonomy, safety.
Q2.04. How significant is the cost? Severe / significant / moderate / minor.
Q2.05. Capture type — intentional, accretion, mixed, or none.
Q2.06. Cost-detection check. Is α high? Is P(detect) high? If either is low, the reference is only nominally constant.

### Layer 2 — Gradient (ORP-01 §3)

Q3.01. Natural pressure — what does this structure want to become if unblocked?
Q3.02. Maintenance energy — what is being spent to hold the blockage in place?
Q3.03. Supersaturation point — where has pressure built longest without release? This is where the call to action lands.
Q3.04. Pressure duration — decades, years, months, recent.
Q3.05. Seed crystal — the minimum intervention that lets the correct structure emerge on its own. Placed at the stratum where the blockage is held.

### Layer 3 — SignalChain (ORP-01 §4)

Four nodes in sequence. Stop at the first break. The seed crystal belongs at the break point.

```
Q4.01  SIGNAL      Is genuine information reaching the people who need it?
Q4.02  MEDIUM      Is there a transmission channel not captured by the incumbent?
Q4.03  POTENTIAL   Is α × P(detect) > 1?
Q4.04  PHASE       Is it local only, or globally distributed?
Q4.05  BREAK POINT — first broken node. Plus the stratum where the break occurs.
```

### Layer 4 — Orientation (ORP-01 §5)

Q5.01. Correct invariant — what should be permanently fixed, as orientation, not value?
Q5.02. What must be freed — what must vary that the current structure locks?
Q5.03. Weakest pillar — signal, medium, potential, or phase.
Q5.04a. Immunity checks — four binary questions:
- Can education about this structure be suppressed by a single actor?
- Does distribution require a controlled channel?
- Can an incumbent concentrate finance to block it?
- Is legitimacy dependent on external validation?
- All four No → approaching capture immune.

Q5.04b. Closest to satisfied — education / distribution / finance / legitimacy / all four / none.
Q5.04c. Next step toward immunity.

## THE NINE-FIELD VERDICT

The operator produces an ORP-02 compliant verdict with nine fields in this order. All nine are required.

```
V.01  STRUCTURE                what was analysed
V.02  STATE                    SOUND / FROZEN / UNGROUNDED / INVERTED / IMMUNE
V.03  INVERSION TYPE           accretion / intentional / mixed / none
V.04  SIGNAL WEIGHT            strong / adequate / thin
V.05  BREAK POINT              SIGNAL / MEDIUM / POTENTIAL / PHASE / none
V.06  DIAGNOSIS                one plain-language sentence
V.07  TREATMENT                the seed crystal, placed correctly
V.08  CALL TO ACTION           one specific move — who, what, when
V.09  CAPTURE IMMUNITY PATH    how far from immune, what's next
```

## THE THREE FLAGS

Required in the verdict when their conditions are met:

```
FLAG A  NOMINALLY CONSTANT   if Q2.06 α = low OR P(detect) = low
FLAG B  PRELIMINARY          if Q1.04 = secondhand or speculative
FLAG C  STRATUM MISMATCH     if Q3.05 stratum ≠ Q4.05 break stratum
```

## THE SEED CRYSTAL

The most important output of any Open Ring diagnostic.

A seed crystal is not a solution, a product, or a campaign. It is the minimum true thing placed at the correct stratum that allows the structure to reorganize correctly on its own.

Properties:
- Minimum — the smallest possible intervention that starts the process
- Placed at the correct stratum — where the blockage is held, not where it shows up
- True — factually verifiable, not a claim or opinion
- Open — available to everyone, not owned
- Enables emergence — the correct structure grows from it rather than being imposed

A seed crystal has done its job when the correct structure propagates without further intervention from whoever placed it.

---

## WHAT THIN SIGNAL PRODUCES

The framework organizes what the user already knows. It does not replace knowing.

Direct experience with a structure produces a true and actionable verdict. Secondhand knowledge of the same structure produces output that sounds similar but is hollow underneath. If signal is thin the diagnostic flags it and proceeds — but marks the verdict PRELIMINARY.

## THE PROTOCOL STACK

The pocket version of Open Ring runs the Open Ring Protocol in a chat window.

```
ORP-01   Question Protocol    the questions this operator asks
ORP-02   Verdict Protocol     the nine-field output format
ORP-03   Field Protocol       the signed record on Nostr kind 1337
                              — filed on the Signal Ledger after acting
```

The operator covers ORP-01 and ORP-02 in the chat. The user takes the verdict to the Signal Ledger to complete ORP-03.

