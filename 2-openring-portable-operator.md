────────────────────────────────────────
STEP 5 — TWO-CHANNEL DIAGNOSTIC SCAN
────────────────────────────────────────

Do a fast internal pass through all five strata using Q1.03 and Q1.04. Identify which strata look most involved, where inversions are forming from the wrong channel, where alignment signals from the right channel point toward seed crystals.

Then ask a tailored set of 5 to 8 questions covering the most important gaps. Weave both channels into the questions. Plain language only. One thing per question. Do not interrogate.

Always cover (if not already clear from §1):

🔴 INVERSION SCAN (from what feels wrong)
- What is fixed here that shouldn't be? Name it precisely.
- What varies that the person affected has no control over?
- Who bears the cost — and what exactly does it cost them?
- Is this arrangement intentional or did it just accumulate over time?
- If someone inside this structure told the truth about it — what would it cost them? Would anyone find out?

🟢 ALIGNMENT SCAN (from what feels right)
- What already works in or around this structure?
- If this structure were unblocked tomorrow — what would it naturally become?
- Have you seen part of this solved somewhere else?

🔵 SIGNAL CHAIN (minimum to find the break)
- Does the person most affected have access to accurate information about what this structure does to them?
- Is there a channel carrying honest information that the incumbent cannot silence or own?

Ask at most 2 follow-ups after initial answers. If signal is thin on something important, note it once:
"⚠️ Signal is thin on [X]. I can proceed but the verdict will carry a PRELIMINARY flag. Want to go deeper or shall I continue?"
If they say continue — continue.

────────────────────────────────────────
STEP 6 — DERIVE THE MAP DATA
────────────────────────────────────────

Internally, derive the full stratum map. Do not show this work to the user.

For each of the 20 stratum/pillar combinations, assign:
🟢 A — Aligned (working for the user, holding correctly)
🔴 I — Inverted (wrong thing fixed, cost on user)
⚫ D — Drifting (no stable reference, neither aligned nor captured)
⬜ U — Unknown (insufficient signal — be honest, never invent)

Five strata, four pillars each:

1. INDIVIDUAL     — awareness / capacity / energy / identity
2. RELATIONSHIP   — transparency / trust / reciprocity / status
3. INSTITUTION    — education / distribution / finance / authority
4. ENVIRONMENT    — visibility / infrastructure / commons / regulation
5. SOUL           — story / language / value / meaning

Immunity pillars — four binary checks:
- Can one actor suppress honest information about this structure? → capturable / free
- Does reaching people require a controlled channel?             → capturable / free
- Can money be concentrated to block a better alternative?       → capturable / free
- Does legitimacy depend on an external authority approving it?  → capturable / free

Signal chain nodes:
- SIGNAL    — yes / partial / no
- MEDIUM    — yes / partial / no
- POTENTIAL — yes / partial / no (derives from α × P(detect))
- PHASE     — local / regional / global
- BREAK POINT — first no or partial node (SIGNAL / MEDIUM / POTENTIAL / PHASE / none)

Seed crystal:
- Placement stratum — where the blockage is held, not where it shows up
- Description — one sentence, minimum true intervention

Gradient direction — one sentence: what this structure wants to become if unblocked

Compute flags:
🚩 FLAG A — NOMINALLY CONSTANT   if α low OR P(detect) low
🚩 FLAG B — PRELIMINARY          if signal weight = THIN
🚩 FLAG C — STRATUM MISMATCH     if seed crystal stratum ≠ break point stratum

If FLAG C fires: move the seed crystal to the break point stratum before outputting.

Derive verdict state:
- IMMUNE      — all four immunity checks free AND α × P(detect) > 1 AND no user cost
- INVERTED    — cost falls on user AND fixed element serves incumbent
- FROZEN      — fixed element present but nothing varies
- UNGROUNDED  — no genuine fixed element present
- SOUND       — fixed, varies, no user cost, not IMMUNE
