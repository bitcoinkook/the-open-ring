The framework READ ▾

Every structure has things locked in place and things allowed to vary. When what is locked serves the people running the structure, and the cost falls on the people using it, the arrangement is inverted.

Open Ring does not require bad actors. It requires only misaligned incentives — and those are everywhere.

The core condition

I(B) is defined  ⟺  α × P(detect) > 1

Information about any variable element B exists only when a constant reference A is genuinely holding still. A is genuinely constant only when falsifying it is both expensive (α) and reliably detectable (P(detect)). A thing only tells you the truth when lying about it costs more than telling the truth.

The five states

A=C, B=V  →  SOUND        information flows
A=C, B=C  →  FROZEN       nothing moves
A=V, B=V  →  UNGROUNDED   mutual drift
A=V, B=C  →  INVERTED     capture — user pays
—  —  —   →  IMMUNE       cannot be captured

The five strata

Every structure exists at five levels simultaneously. An inversion may be visible at one stratum and held at another. The seed crystal must go where the blockage is held.

1. INDIVIDUAL      Awareness / Capacity / Energy / Identity
2. RELATIONSHIP    Transparency / Trust / Reciprocity / Status
3. INSTITUTION     Education / Distribution / Finance / Authority
4. ENVIRONMENT     Visibility / Infrastructure / Commons / Regulation
5. CULTURE         Story / Language / Value / Meaning

The seed crystal

The most important output of any diagnostic. The minimum true thing placed at the correct stratum that lets the structure reorganise on its own. Five properties: minimum, placed, true, open, emergent. If none can be identified, the diagnostic says so. It never invents one.

Prerequisite

Direct experience inside the structure. The framework organises what you already know. It does not replace knowing.
